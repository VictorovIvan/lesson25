Используется Pattern factory. Позволяет добавлять дополнительную функцию для тестирования программы.
Тестовый режим заключается в вводе "test" - без авторизации, любое значение имени и пароля,
для дальнейшего входа в программу. Заходит как пользователь Smith Stewart.
либо "product" - требуется ввести значение имя-пароль(день рождения)
